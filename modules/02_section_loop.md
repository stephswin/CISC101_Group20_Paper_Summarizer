# Module 2: Section Loop

Iterate through each section.  
Summarize content while respecting length constraints.  
Flag sections <50 words or empty.

---

## Summary Level Mode (New Requirement A)

Add variable: **`summary_level` ∈ {"short", "detailed"}`**

- **If `summary_level = "short"`**
  - Produce a **1–2 sentence summary** for the section.

- **If `summary_level = "detailed"`**
  - Produce:
    - A **short paragraph** summary.
    - A **bullet list of 3–5 key points** from the section.
